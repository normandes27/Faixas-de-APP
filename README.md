# Faixas-de-APP
Gerador automatizado de faixas de Áreas de Preservação Permanente no QGIS 
Projeto PRAD Toolbox (LabGEO UFR - LabGIS UFMS) 


Como instalar: a) abra o programa QGIS 3x; b) clique em "Complementos"; c) clique em "Gerenciar e Instalar Complementos"; d) clique em "Instalar a partir do ZIP"; e) clique em Procurar arquivo ZIP no computador ==>  [...]; f) selecione o arquivo zip desejado; g) clique em "Instalar Complemento"; 


Vídeo explicativo sobre o plugin: https://drive.google.com/file/d/1dB0x-LvbRfgKnbaH2B7gw7hOprV-lfZg/view?usp=sharing (gerado por IA - NotebookLM - https://notebooklm.google.com/)
